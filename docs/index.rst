Welcome to UnixV5 Modernized's documentation!
=============================================

This project aims to modernize the Research Unix Version 5 codebase for study and potential execution on x86_64 systems. It involves:

*   Refactoring C code to modern standards.
*   Implementing a CMake-based build system.
*   Generating comprehensive documentation using Doxygen and Sphinx.
*   Porting components to the x86_64 architecture.
